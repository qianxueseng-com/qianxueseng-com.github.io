which os for front-end

还是用Mac或者Ubuntu吧

习惯了看到所有的程序员都用Mac，所以很诧异的看到有一半的人都还在用Windows。参看这个对于前端学员的投票。

用Mac或者Ubuntu的好处

  • 熟悉命令行
  • 方便使用node.js。node.js在windows上问题多多。
  • 开发环境和production环境更接近

如果现在在用windows，可以考虑

  • 安装虚拟机，例如virtualbox
  • 在虚拟机里面装Ubuntu
